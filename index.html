Practical Carpenter Rules (TEK17)

Quick On-Site Method:
  1. Measure Total Height (H).
  2. Divide by ~150-160mm (Outdoor) or 170 (Indoor). Round.
  3. Calc Exact Rise = H / Steps.
  4. Calc Depth = 630 - 2*Rise.

Blondel Formula: 2 x Rise + Run = 620 - 640 mm
Comfort rule based on average human stride (~63cm).

  • Private (Indoor): h≤180mm, d≥250mm
  • Outdoor (Terrace): h=130-160mm, d=280-320mm
  • Handrail: Stairs=900mm, Terrace≥1000mm
